# Task 6: World Happiness Visualization

This project analyzes the 2019 World Happiness Report `2019.csv` to visualize global happiness trends and correlations using **pure Matplotlib** and **Pandas**.

## Files
- `Task6_Visualization.ipynb`: Main Jupyter Notebook with code, plots, and analysis.
- `insights.md`: Detailed breakdown of the analytical findings.
- `2019.csv`: Dataset used for analysis.

## Dependencies
- python 3.x
- pandas
- matplotlib
- jupyter (to open the notebook)

## Visualizations Included
1.  **Bar Chart**: Top 10 Happiest Countries.
2.  **Stacked Bar Chart**: Breakdown of happiness factors (GDP, Support, etc.).
3.  **Horizontal Bar Chart**: Top 10 vs Bottom 10 comparison.
4.  **Histogram**: Distribution of Happiness Scores.
5.  **Box Plot**: Distribution variance of key numerical factors.
6.  **Correlation Heatmap**: Visualizing correlation coefficients.
7.  **Scatter Matrix**: Pairwise correlations between Score, GDP, Support, and Health.
8.  **Bubble Plot**: GDP vs. Happiness (sized by Generosity).
9.  **Parallel Coordinates**: Multivariate profile of happiness tiers (Low/Med/High).

## How to Run
1.  **Open the Notebook**:
    ```bash
    jupyter notebook Task6_Visualization.ipynb
    ```
    or upload `Task6_Visualization.ipynb` to [Google Colab](https://colab.research.google.com/).
    
2.  **Execute Cells**: Run all cells to generate the charts inline.
